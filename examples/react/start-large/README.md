# TanStack Start - Large Example

A large-scale TanStack Start example demonstrating performance with many routes.

- [TanStack Router Docs](https://tanstack.com/router)

## Start a new project based on this example

To start a new project based on this example, run:

```sh
npx gitpick TanStack/router/tree/main/examples/react/start-large start-large
```

## Getting Started

From your terminal:

```sh
pnpm install
pnpm dev
```

This starts your app in development mode, rebuilding assets on file changes.

## Build

To build the app for production:

```sh
pnpm build
```

## About This Example

This example contains a large number of routes to demonstrate TanStack Router's performance and scalability with large applications.
