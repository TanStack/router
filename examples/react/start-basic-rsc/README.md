# TanStack Start - RSC Example

This site is built with TanStack Router and React Server Components!

- [TanStack Router Docs](https://tanstack.com/router)

## Start a new project based on this example

To start a new project based on this example, run:

```sh
npx gitpick TanStack/router/tree/main/examples/react/start-basic-rsc start-basic-rsc
```

## Deployment

It's deployed automagically with Netlify!

- [Netlify](https://netlify.com/)

## Getting Started

From your terminal:

```sh
pnpm install
pnpm dev
```

## Build

Build for production:

```sh
pnpm build
```

## About This Example

This example demonstrates:

- React Server Components (RSC) with TanStack Start
- Server-side rendering
